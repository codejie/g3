# G3 Project
G3工程是一个OpenCode的增强版本，旨在提供更强大的功能和更好的用户体验。以下是G3工程的主要特点：
- 支持多用户同时使用OpenCode服务
- 支持会话管理，可以保存和恢复用户的会话状态
- 支持用户数据隔离，通过目录结构和权限控制，实现用户只能访问自己的数据
- 支持文件上传和下载，方便用户管理自己的文件
- 支持日志记录，实现管理员监控和调试系统
- 提供扩展服务和API接口，实现对OpenCode的功能扩展，包括模型配置、Skills部署等
- 提供Web界面，实现用户进行操作和服务管理


# 工程组成
G3工程是由两个独立的服务组成的，分别是前端服务和后端服务。

## 前端服务/frontend
前端服务提供Web页面服务，用户通过Web界面实现OpenCode的功能操作、扩展服务和配置、用户管理。
### 技术栈
- Typescript 5.x: 作为主要的编程语言
- Vue.js 3.5.x: 作为前端框架，提供响应式和组件化的开发方式
- Vite 8.x: 作为前端构建工具，提供快速的开发和构建
- Element Plus 2.x: 作为UI组件库，提供丰富的组件和良好的设计风格
- Vue Router 5.x: 作为路由管理，支持SPA应用
- Pinia 3.x: 作为状态管理，提供响应式数据存储
- i18next 26.x: 作为国际化库，提供多语言支持和灵活的翻译管理
- Vue I18n 11.x: Vue集成i18next
- Axios: 作为HTTP客户端，用于API请求

### 已实现页面
- Login.vue: 用户登录页面，提供用户名密码登录
- Home.vue: 用户主页面，类似ChatGPT聊天界面，包含侧边栏和消息区域
- Admin.vue: 管理员页面，提供用户管理、日志查看、系统配置功能入口


## 后端服务/backend
后端服务提供API接口服务，处理前端请求，访问数据库，管理用户会话和文件操作。
### 技术栈
- Typescript 5.8.3: 作为主要的编程语言
- Fastify 5.8.5: 作为后端框架，提供高性能和丰富的生态系统
- better-sqlite3 12.9.0: 作为数据库驱动，提供轻量级和易于使用的数据库解决方案
- uuid 13.0.0: 作为唯一标识符生成库
### 功能设计
- 用户认证：提供用户登录、注册、权限验证等功能
- 会话管理：管理用户会话状态，支持会话创建、恢复、删除
- 文件管理：处理文件上传、下载、删除等操作
- 日志记录：记录系统操作日志，支持管理员查询
- API服务：提供RESTful API接口供前端调用
### 数据库设计
- users表：存储用户的基本信息，包括用户名、密码、角色（user/admin）、创建时间等。
- projects表：存储用户的项目信息，包括项目ID、用户ID、项目名称、项目描述、创建时间等。
- sessions表：存储用户的会话信息，包括会话ID、用户ID、项目ID、会话状态、创建时间等。
- logs表：存储系统的日志信息，包括日志类型、日志内容、用户ID、创建时间等。
- files表：存储用户上传、下载的文件信息，包括文件名称、文件路径、文件类型、文件大小、创建时间等。
